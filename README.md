# SCP-087-B Godot
This is a port of the original [SCP-087-B](https://www.scpcbgame.com/scp-087-b.html) to [Godot Engine](https://godotengine.org/) (GDScript)

I did this port for fun, it's not very high quality though. *(Pull Requests welcome!)*

This port features:

 - Main menu
 - Settings menu
 - Pause menu
 - Debug menu *(See more info [here](https://github.com/SlendyMawn/scp-087-b-godot/wiki/Debug-Menu))*
 - Uncapped framerate
 - Android version
 - Web version
 - Gamepad support *(Tested with [Steam Controller](https://store.steampowered.com/hardware/steamcontroller))*

Feel free to fork this and build off of it as you wish, but *[credit would also be nice!](https://www.youtube.com/@SlendyMawn)*
